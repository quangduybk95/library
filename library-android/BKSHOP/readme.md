# BK SHOP 
BK SHOP というアプリケーションは 小規模店舗のオンライン販売向けのアプリです。
大規模なシステムと競合することができませんので、私は小さなお店にむけました。
商品のデータベースだけがいります。どんな商品でも売られます。
